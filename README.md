# Java Student Management System

## Overview
A console-based Student Management System developed using Java and Object-Oriented Programming (OOP) concepts.

## Features
- Add Student
- View All Students
- Search Student by ID
- Update Student Details
- Delete Student Records
- Menu-Driven Interface
- Scanner-Based User Input
- Exception Handling
- Input Validation
- File Handling
- Data Persistence
- JDBC Connectivity
- MySQL Database Integration

## Technologies Used
- Java
- JDBC
- MySQL
- ArrayList
- File Handling
## Concepts Used
- Classes and Objects
- Encapsulation
- ArrayList Collection
- CRUD Operations
- OOP Concepts
